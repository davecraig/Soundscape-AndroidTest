# Om Soundscape

Soundscape utforsker bruken av nyskapende lydbasert teknologi for å hjelpe folk med å få en rikere bevissthet om omgivelsene sine, slik at de blir mer selvsikre og i stand til å ta seg rundt.

Soundscape ville ikke vært mulig uten disse fantastiske tredjepartsbibliotekene og -dataene. Klikk på lenkene for å se lisensene deres:

[©OpenStreetMap contributors](https://www.openstreetmap.org/copyright) leverer alle geodataene som brukes til kartlegging og lyd i Soundscape.

[©OpenMapTiles](https://github.com/openmaptiles/openmaptiles/blob/master/LICENSE.md) brukes til å generere kartflisene vi bruker.

[Steam Audio library ©Valve Corporation](https://github.com/ValveSoftware/steam-audio) brukes til å generere 3D-lyden, og er gjort tilgjengelig for bruk under [Apache 2.0-lisensen](https://github.com/ValveSoftware/steam-audio/blob/master/LICENSE.md).

Alle tredjeparts produktnavn, firmanavn og logoer er varemerker eller registrerte varemerker og tilhører sine respektive eiere.
