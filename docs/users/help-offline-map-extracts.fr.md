# Extraits de carte hors ligne

Soundscape peut télécharger des régions de données cartographiques sur votre téléphone afin de continuer à fonctionner lorsque vous n’avez pas de connexion Internet. La carte ci-dessous indique les régions actuellement disponibles au téléchargement. Elles existent en trois variantes :

- **Pays** — le pays entier en un seul extrait.
- **Région** — un état, une province ou une autre subdivision de premier niveau d’un pays.
- **Ville** — un petit extrait couvrant une ville et les localités qui l’entourent immédiatement.

Touchez n’importe quel endroit de la carte pour voir tous les extraits qui couvrent ce point — un pays, la région qui s’y trouve et tout extrait de ville sont répertoriés ensemble, avec le même nom, la même taille et la même liste de localités que dans l’écran **Cartes hors ligne** de l’application.
