# سبک‌های جهت‌نما

ساند‌اسکیپ به شما امکان می‌دهد در
**تنظیمات → صدا → سبک‌های جهت‌نمای صوتی** میان چندین صدای متفاوت جهت‌نما انتخاب کنید. همه‌ی آنها یک کار را انجام می‌دهند —
کشاندن شما به سمت یک مقصد با استفاده از صدای فضایی سه‌بُعدی — اما صدای آنها
بسیار با یکدیگر متفاوت است.

هر جهت‌نما از یک مجموعه‌ی کوچک از نمونه‌های تکرارشونده ساخته شده است. ساند‌اسکیپ
به‌طور خودکار نمونه‌ی درست را برای زاویه‌ی میان جهتی که رو به آن ایستاده‌اید
و جهت جهت‌نما انتخاب می‌کند، بنابراین با چرخاندن سرتان یا
حرکت کردن، صدا تغییر می‌کند. نمونه‌های «روی محور» زمانی پخش می‌شوند که رو به جهت‌نما نشانه گرفته باشید؛ نمونه‌های
«پشت‌سر» زمانی پخش می‌شوند که جهت‌نما پشت سر شما باشد.

## چگونه از این صفحه استفاده کنیم

برای هر جهت‌نمای زیر، این موارد را خواهید یافت:

- **یک نمونه‌ی صوتی تکرارشونده** که هر یک از حالت‌های زاویه‌ای را به‌نوبت پخش می‌کند،
  از صدای روی محور (در پیش رو) آغاز می‌شود و با صدای پشت‌سر
  پایان می‌یابد، تا بتوانید بشنوید که جهت‌نما هنگام چرخش شما چگونه تغییر می‌کند.
- **یک نمودار طیف فرکانسی** که نشان می‌دهد انرژی جهت‌نما در کجای
  بازه‌ی شنیداری قرار دارد. اگر در فرکانس‌های خاصی دچار افت شنوایی هستید،
  جهت‌نماهایی که بیشتر انرژی‌شان در نواحی‌ای است که خوب می‌شنوید،
  دنبال کردنشان آسان‌تر خواهد بود.

نمودارها به‌گونه‌ای نرمال‌سازی شده‌اند که بلندترین قله روی ۰&nbsp;dB قرار می‌گیرد؛ این منحنی‌ها
برای مقایسه‌ی *شکل* محتوای فرکانسی هر جهت‌نما مفیدند، نه بلندی
مطلق آن.

**توجه:** نمونه‌های صوتی اینجا تک‌کانالی (مونو) هستند و از فضاسازی سه‌بُعدی استفاده نمی‌کنند.
در خودِ اپلیکیشن صدا به‌صورت دوگوشی (binaural) رندر می‌شود، بنابراین وقتی هدفون استریو می‌پوشید،
به نظر می‌رسد که از یک جهت مشخص می‌آید.

<script>
  /* Only allow one beacon audio sample to play at a time. Kramdown strips
     newlines inside <script> blocks, so a // comment would swallow the rest
     of the script — use a block comment. */
  document.addEventListener("DOMContentLoaded", function () {
    var players = document.querySelectorAll("audio");
    players.forEach(function (player) {
      player.addEventListener("play", function () {
        players.forEach(function (other) {
          if (other !== player) {
            other.pause();
          }
        });
      });
    });
  });
</script>
