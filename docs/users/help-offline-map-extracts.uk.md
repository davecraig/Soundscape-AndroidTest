# Офлайн-фрагменти карти

Soundscape може завантажувати на телефон регіони з картографічними даними, щоб додаток продовжував працювати, коли немає інтернет-з’єднання. На карті нижче показано регіони, доступні для завантаження зараз. Вони бувають трьох видів:

- **Країна** — уся країна одним фрагментом.
- **Регіон** — штат, провінція або інша одиниця адміністративного поділу країни найвищого рівня.
- **Місто** — невеликий фрагмент, що охоплює місто та безпосередньо прилеглі населені пункти.

Торкніться будь-якого місця на карті, щоб побачити всі фрагменти, що охоплюють цю точку — країна, регіон у ній та будь-який міський фрагмент перелічуються разом, з тією самою назвою, розміром і списком населених пунктів, що й на екрані **Офлайн-карти** у додатку.
