# Extractos de mapa sin conexión

Soundscape puede descargar regiones de datos de mapas en tu teléfono para que siga funcionando cuando no tengas conexión a Internet. El mapa de abajo muestra las regiones que están disponibles actualmente para descargar. Hay tres variantes:

- **País** — el país entero en un solo extracto.
- **Región** — un estado, provincia u otra subdivisión de primer nivel de un país.
- **Ciudad** — un extracto pequeño que cubre una ciudad y las localidades inmediatamente cercanas.

Toca cualquier punto del mapa para ver todos los extractos que cubren ese punto: un país, la región que contiene y cualquier extracto de ciudad aparecen juntos, con el mismo nombre, tamaño y lista de localidades que se muestran en la pantalla **Mapas sin conexión** de la aplicación.
