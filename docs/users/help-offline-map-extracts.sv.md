# Offline-kartutdrag

Soundscape kan ladda ner regioner med kartdata till din telefon så att appen fortsätter att fungera när du inte har någon internetanslutning. Kartan nedan visar de regioner som för närvarande är tillgängliga för nedladdning. De finns i tre varianter:

- **Land** — hela landet i ett enda utdrag.
- **Region** — en delstat, provins eller annan överordnad indelning av ett land.
- **Stad** — ett litet utdrag som täcker en stad och orterna runt omkring.

Tryck var som helst på kartan för att se alla utdrag som täcker den punkten — ett land, regionen i det och eventuella stadsutdrag visas tillsammans, med samma namn, storlek och ortlista som på skärmen **Offline-kartor** i appen.
