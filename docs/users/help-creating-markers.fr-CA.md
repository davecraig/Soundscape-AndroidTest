# Création de marqueurs

Vous pouvez créer des marqueurs de trois manières : en recherchant l’endroit à enregistrer avec la barre de recherche, en le trouvant via le bouton *« Emplacements à proximité »* ou en utilisant le bouton *« Emplacement actuel »* — tous ces boutons se trouvent sur l’écran d’accueil de Soundscape. Une fois l’emplacement choisi, sa sélection vous amènera à l’écran *« Détails de l’emplacement »*. Sur cet écran, sélectionnez le bouton *« Enregistrer en tant que marqueur »*.

Vous avez maintenant la possibilité de personnaliser ce marqueur. Si vous le souhaitez, vous pouvez modifier le nom du marqueur et ajouter une annotation qui sera appelée avec le marqueur pour fournir des informations supplémentaires. Une fois que vous avez terminé, sélectionnez le bouton *« Terminé »* pour enregistrer votre marqueur.
