# Criar Marcos

Pode criar marcos de três formas: pesquisando o local que pretende guardar usando a barra de pesquisa, encontrando um local através do botão *"Locais nas Proximidades"*, ou usando o botão *"Localização Atual"*, todos disponíveis no ecrã principal do Soundscape. Depois de encontrar o local desejado, selecioná‑lo levar‑á ao ecrã *"Detalhes da Localização"*. Nesse ecrã, selecione o botão *"Guardar como Marco"*.

Agora terá a opção de personalizar este marco. Pode alterar o nome do marco e também adicionar uma anotação que será anunciada juntamente com o marco para fornecer informação adicional. Quando terminar, selecione o botão *"OK"* para guardar o seu marco.
