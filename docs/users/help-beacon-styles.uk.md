# Стилі маячка

Soundscape дозволяє вибрати один із кількох різних звуків маячка в розділі
**Налаштування → Аудіо → Стилі звукового маячка**. Усі вони виконують однакову роботу —
тягнуть вас до пункту призначення за допомогою тривимірного просторового аудіо — але звучать
дуже по-різному.

Кожен маячок побудований із невеликого набору зациклених семплів. Soundscape
автоматично обирає правильний семпл для кута між напрямком, у якому ви дивитеся,
та напрямком маячка, тож звук змінюється, коли ви повертаєте голову або
рухаєтеся. Семпли «на осі» відтворюються, коли ви спрямовані на маячок; семпли
«ззаду» відтворюються, коли він позаду вас.

## Як користуватися цією сторінкою

Для кожного маячка нижче ви знайдете:

- **Зациклений аудіосемпл**, який по черзі відтворює кожен з варіантів за кутом,
  починаючи зі звуку «на осі» (попереду) і закінчуючи звуком «ззаду»,
  щоб ви могли почути, як маячок змінюється, коли ви повертаєтеся.
- **Графік частотного спектра**, що показує, де лежить енергія маячка в
  чутному діапазоні. Якщо у вас зниження слуху на певних частотах,
  маячки, більшість енергії яких розташована в регіонах, які ви добре чуєте, буде
  найлегше відстежувати.

Графіки нормалізовані так, що найгучніший пік розташований на 0&nbsp;дБ; криві
корисні для порівняння *форми* частотного вмісту кожного маячка, а не його
абсолютної гучності.

**Примітка:** аудіосемпли тут монофонічні й не використовують тривимірну просторову обробку.
У самому застосунку звук відтворюється бінаурально, тож він здаватиметься таким,
що лунає з певного напрямку, коли ви носите стереонавушники.

     of the script — use a block comment. */
  document.addEventListener("DOMContentLoaded", function () {
    var players = document.querySelectorAll("audio");
    players.forEach(function (player) {
      player.addEventListener("play", function () {
        players.forEach(function (other) {
          if (other !== player) {
            other.pause();
          }
        });
      });
    });
  });
