# オフライン地図エクストラクト

Soundscape は地図データの地域を端末にダウンロードできるため、インターネット接続がないときでも動作し続けます。下の地図は、現在ダウンロードできる地域を示しています。次の3種類があります:

- **国** — 国全体を1つのエクストラクトにまとめたものです。
- **地域** — 国の州、県、その他の最上位の行政区分です。
- **都市** — 1つの都市とそのすぐ周辺の町をカバーする小さなエクストラクトです。

地図上の任意の場所をタップすると、その地点をカバーするすべてのエクストラクトが表示されます。国、その中の地域、そこにある都市のエクストラクトがまとめて一覧表示され、アプリの **オフライン地図** 画面と同じ名前・サイズ・地名リストが表示されます。
