# Voices

Soundscape can use any of the voices you have downloaded to your Android phone in the Text-to-speech output settings. Within the Soundscape app, to select a voice other than the default Android voice, go to *"Settings"* from the main menu, expand the *"Audio"* section and then select *"Voices"*.
