# Χρήση στοιχείων ελέγχου πολυμέσων

## Τι είναι?

Μπορείτε να αποκτήσετε πρόσβαση σε ορισμένες λειτουργίες του Soundscape με τη βοήθεια των κουμπιών ελέγχου πολυμέσων στα ακουστικά σας. Αυτή η λειτουργία λειτουργεί με ενσύρματα ακουστικά ή ακουστικά Bluetooth που έχουν κουμπιά χειριστήριων πολυμέσων όπως Αναπαραγωγή, Παύση, Επόμενο, Προηγούμενο και άλλα. Διαφορετικά ακουστικά μπορεί να περιλαμβάνουν διαφορετικά κουμπιά, οπότε ανατρέξτε στη λίστα των παρακάτω ενεργειών για να προσδιορίσετε ποια είναι διαθέσιμα σε εσάς.Θυμηθείτε επίσης ότι αυτή η λειτουργία λειτουργεί μόνο με ακουστικά που υποστηρίζουν τα στοιχεία ελέγχου πολυμέσων της Apple (όπως αναπαραγωγή και παύση).

## Πότε θα το χρησιμοποιούσα;

Τα χειριστήρια πολυμέσων ακουστικών μπορούν να χρησιμοποιηθούν κατά την εκτέλεση του Soundscape. Αυτό ισχύει είτε βρίσκεστε στο Soundscape είτε ενώ το Soundscape βρίσκεστε στο παρασκήνιο, και ακόμη και όταν η συσκευή σας είναι κλειδωμένη. Θυμηθείτε, ωστόσο, ότι τα κουμπιά χειριστήριων πολυμέσων ακουστικών δεν θα λειτουργούν στο Soundscape αν αναπαράγετε ήχο όπως μουσική, podcast. βίντεο με άλλη εφαρμογή.

## Πώς λειτουργεί;

Υπάρχουν 3 λειτουργίες για τα στοιχεία ελέγχου πολυμέσων. Η λειτουργία μπορεί να επιλεγεί στην ενότητα *"Ρυθμίσεις"* *"Στοιχεία ελέγχου πολυμέσων"*. Οι λειτουργίες είναι:

Αρχική λειτουργία.

⏯ Αναπαραγωγή/Παύση: Εναλλάσσει τον ήχο του ηχητικού σήματος (ενεργοποίηση/απενεργοποίηση).

⏭ Επόμενο: Εάν αναπαράγεται μια διαδρομή, μετατοπίζει το ηχητικό σήμα στο επόμενο σημείο πορείας της διαδρομής. Εάν δεν αναπαράγεται καμία διαδρομή, ανακοινώνει *"Η τοποθεσία μου"*.

⏮ Προηγούμενο: Εάν αναπαράγεται μια διαδρομή, μετατοπίζει το ηχητικό σήμα στο προηγούμενο σημείο πορείας της διαδρομής. Εάν δεν αναπαράγεται καμία διαδρομή, ανακοινώνει *"Γύρω μου"*.



Φωνητικές εντολές.



⏯ Αναπαραγωγή/Παύση: Ξεκινά την ακρόαση για φωνητικές εντολές.



Ηχητικό μενού.



⏭ Επόμενο και ⏮ Προηγούμενο: Μετακινούνται σε μια σειρά επιλογών μενού που η εφαρμογή περιγράφει με σύνθεση ομιλίας. ⏯ Αναπαραγωγή/Παύση: Εκτελεί την επιλογή που περιγράφεται. Υπάρχει ένα κορυφαίο μενού (top level) που περιέχει μόνο υπο-μενού ως επιλογές. Κάθε υπο-μενού περιέχει μια ομάδα συναφών ενεργειών.
