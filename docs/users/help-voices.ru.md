# Голоса

Soundscape может использовать любые голоса, которые вы загрузили на телефон Android в настройках *"Преобразование текста в речь"*. В приложении Soundscape, чтобы выбрать голос, отличный от системного голоса Android, откройте *"Настройки"* в главном меню, разверните раздел *"Аудио"* и выберите *"Голоса"*.
