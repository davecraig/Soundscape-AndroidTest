# 离线地图区块

Soundscape 可以将地图数据的区域下载到您的手机上，以便在没有网络连接时仍能继续使用。下面的地图显示了当前可供下载的区域。它们分为三种：

- **国家** — 整个国家作为一个区块。
- **地区** — 一个国家的州、省或其他最高级别的行政区划。
- **城市** — 覆盖一座城市及其紧邻城镇的小型区块。

点按地图上的任意位置，即可查看覆盖该点的所有区块——国家、其中的地区以及该处的任何城市区块都会一起列出，并显示与应用中 **离线地图** 屏幕相同的名称、大小和城镇列表。
