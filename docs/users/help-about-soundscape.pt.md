# Acerca do Soundscape

Soundscape explora o uso de tecnologia inovadora baseada em áudio para permitir que as pessoas construam uma consciência mais rica de seus arredores, tornando-se assim mais confiante e capacitado para se locomover.

O Soundscape não seria possível sem estas fantásticas bibliotecas e dados de terceiros. Clique nas ligações para ver as respetivas licenças:

[©OpenStreetMap contributors](https://www.openstreetmap.org/copyright) fornecem todos os dados geográficos para o mapeamento e o áudio utilizados no Soundscape.

[©OpenMapTiles](https://github.com/openmaptiles/openmaptiles/blob/master/LICENSE.md) é usado para gerar os tiles de mapa que utilizamos.

[Steam Audio library ©Valve Corporation](https://github.com/ValveSoftware/steam-audio) é usado para gerar o áudio 3D e encontra‑se disponível para utilização ao abrigo da [licença Apache 2.0](https://github.com/ValveSoftware/steam-audio/blob/master/LICENSE.md).

Todos os nomes de produtos e empresas de terceiros, bem como os respetivos logótipos, são marcas comerciais ou marcas registadas e continuam a ser propriedade dos respetivos titulares.
