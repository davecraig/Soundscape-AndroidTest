# Offline-Kartenausschnitte

Soundscape kann Regionen mit Kartendaten auf Ihr Telefon herunterladen, damit es auch ohne Internetverbindung weiter funktioniert. Die Karte unten zeigt die Regionen, die derzeit zum Herunterladen verfügbar sind. Es gibt sie in drei Varianten:

- **Land** — das gesamte Land in einem einzigen Ausschnitt.
- **Region** — ein Bundesland, eine Provinz oder eine andere übergeordnete Verwaltungseinheit eines Landes.
- **Stadt** — ein kleiner Ausschnitt, der eine Stadt und die unmittelbar umliegenden Orte abdeckt.

Tippen Sie auf eine beliebige Stelle der Karte, um alle Ausschnitte zu sehen, die diesen Punkt abdecken — ein Land, die darin liegende Region und alle Stadtausschnitte werden zusammen aufgeführt, mit demselben Namen, derselben Größe und derselben Ortsliste wie im Bildschirm **Offline-Karten** in der App.
