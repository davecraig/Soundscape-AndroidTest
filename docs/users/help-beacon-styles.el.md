# Στυλ ηχητικού σήματος

Το Soundscape σάς επιτρέπει να επιλέξετε ανάμεσα σε αρκετούς διαφορετικούς ήχους ηχητικού σήματος στις
**Ρυθμίσεις → Ήχος → Στυλ ηχητικού σήματος**. Όλα κάνουν την ίδια δουλειά —
σας τραβούν προς έναν προορισμό χρησιμοποιώντας χωρικό ήχο 3D — αλλά ακούγονται
πολύ διαφορετικά μεταξύ τους.

Κάθε ηχητικό σήμα είναι φτιαγμένο από ένα μικρό σύνολο δειγμάτων που επαναλαμβάνονται σε βρόχο. Το Soundscape
επιλέγει αυτόματα το σωστό δείγμα για τη γωνία ανάμεσα στην κατεύθυνση που είστε στραμμένοι
και την κατεύθυνση του ηχητικού σήματος, έτσι ώστε ο ήχος να αλλάζει καθώς γυρίζετε το κεφάλι σας ή
μετακινείστε. Τα δείγματα "επί του άξονα" αναπαράγονται όταν είστε στραμμένοι προς το ηχητικό σήμα· τα
δείγματα "πίσω" αναπαράγονται όταν αυτό βρίσκεται πίσω σας.

## Πώς να χρησιμοποιήσετε αυτή τη σελίδα

Για κάθε ηχητικό σήμα παρακάτω θα βρείτε:

- **Ένα δείγμα ήχου σε βρόχο** που αναπαράγει με τη σειρά κάθε μία από τις παραλλαγές γωνίας,
  ξεκινώντας από τον ήχο επί του άξονα (μπροστά) και τελειώνοντας με τον ήχο πίσω σας,
  ώστε να μπορείτε να ακούσετε πώς αλλάζει το ηχητικό σήμα καθώς περιστρέφεστε.
- **Ένα γράφημα φάσματος συχνοτήτων** που δείχνει πού βρίσκεται η ενέργεια του ηχητικού σήματος μέσα
  στο ακουστό εύρος. Εάν έχετε απώλεια ακοής σε συγκεκριμένες συχνότητες,
  τα ηχητικά σήματα με το μεγαλύτερο μέρος της ενέργειάς τους στις περιοχές που ακούτε καλά θα είναι
  τα πιο εύκολα να ακολουθήσετε.

Τα γραφήματα είναι κανονικοποιημένα ώστε η πιο δυνατή κορυφή να βρίσκεται στα 0&nbsp;dB· οι καμπύλες είναι
χρήσιμες για τη σύγκριση του *σχήματος* του περιεχομένου συχνοτήτων κάθε ηχητικού σήματος, όχι της
απόλυτης έντασής του.

**Σημείωση:** τα δείγματα ήχου εδώ είναι μονοφωνικά και δεν χρησιμοποιούν χωρική απόδοση 3D.
Στην ίδια την εφαρμογή ο ήχος αποδίδεται διωτικά, οπότε θα φαίνεται ότι
προέρχεται από μια συγκεκριμένη κατεύθυνση όταν φοράτε στερεοφωνικά ακουστικά.

<script>
  /* Only allow one beacon audio sample to play at a time. Kramdown strips
     newlines inside <script> blocks, so a // comment would swallow the rest
     of the script — use a block comment. */
  document.addEventListener("DOMContentLoaded", function () {
    var players = document.querySelectorAll("audio");
    players.forEach(function (player) {
      player.addEventListener("play", function () {
        players.forEach(function (other) {
          if (other !== player) {
            other.pause();
          }
        });
      });
    });
  });
</script>
