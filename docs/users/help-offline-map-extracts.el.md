# Αποσπάσματα χάρτη εκτός σύνδεσης

Το Soundscape μπορεί να κατεβάσει περιοχές δεδομένων χάρτη στο τηλέφωνό σας ώστε να συνεχίζει να λειτουργεί όταν δεν έχετε σύνδεση στο διαδίκτυο. Ο παρακάτω χάρτης δείχνει τις περιοχές που είναι διαθέσιμες για λήψη αυτή τη στιγμή. Διατίθενται σε τρεις μορφές:

- **Χώρα** — ολόκληρη η χώρα σε ένα μόνο απόσπασμα.
- **Περιφέρεια** — μια πολιτεία, επαρχία ή άλλη ανώτατου επιπέδου υποδιαίρεση μιας χώρας.
- **Πόλη** — ένα μικρό απόσπασμα που καλύπτει μια πόλη και τους άμεσα γειτονικούς οικισμούς.

Πατήστε οπουδήποτε στον χάρτη για να δείτε κάθε απόσπασμα που καλύπτει αυτό το σημείο — μια χώρα, η περιφέρεια εντός της και οποιοδήποτε απόσπασμα πόλης εμφανίζονται μαζί, με το ίδιο όνομα, μέγεθος και λίστα οικισμών που εμφανίζονται στην οθόνη **Χάρτες εκτός σύνδεσης** της εφαρμογής.
