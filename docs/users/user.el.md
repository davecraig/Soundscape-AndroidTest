Η τεκμηρίωση εδώ προέρχεται από την πιο πρόσφατη έκδοση της εφαρμογής. Στην εφαρμογή μπορείτε να τη βρείτε κάνοντας κλικ στο κουμπί του μενού (επάνω αριστερά) και στη συνέχεια επιλέγοντας «Βοήθεια και Οδηγοί».

Για όσους ενδιαφέρονται για το πώς λειτουργεί η εφαρμογή, ρίξτε μια ματιά σε [αυτή τη σελίδα]({{ "/users/how-it-works.html" | relative_url }}).
