# Offline-karttaotteet

Soundscape voi ladata karttatietojen alueita puhelimeesi, jotta se toimii edelleen, kun sinulla ei ole internetyhteyttä. Alla oleva kartta näyttää alueet, jotka ovat tällä hetkellä ladattavissa. Niitä on kolmea tyyppiä:

- **Maa** — koko maa yhtenä otteena.
- **Alue** — maan osavaltio, provinssi tai muu ylimmän tason hallinnollinen jako.
- **Kaupunki** — pieni ote, joka kattaa kaupungin ja sen välittömässä läheisyydessä olevat taajamat.

Napauta mitä tahansa kohtaa kartalla nähdäksesi kaikki kyseisen pisteen kattavat otteet — maa, sen sisällä oleva alue ja mahdolliset kaupunkiotteet luetellaan yhdessä, samalla nimellä, koolla ja paikkakuntaluettelolla, jotka näkyvät sovelluksen **Offline-kartat**-näytöllä.
