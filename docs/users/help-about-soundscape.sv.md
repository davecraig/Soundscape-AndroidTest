# Om Soundscape

Soundscape utforskar användningen av innovativ ljudbaserad teknik för att göra det möjligt för människor att bygga upp en djupare medvetenhet om sin omgivning, och därmed bli mer självsäkra och stärkta i att ta sig fram.

Soundscape skulle inte vara möjligt utan dessa fantastiska tredjepartsbibliotek och data. Klicka på länkarna för att se deras licenser:

[©OpenStreetMap contributors](https://www.openstreetmap.org/copyright) tillhandahåller all geodata för kartläggning och ljud som används i Soundscape.

[©OpenMapTiles](https://github.com/openmaptiles/openmaptiles/blob/master/LICENSE.md) används för att generera de kartplattor vi använder.

[Steam Audio library ©Valve Corporation](https://github.com/ValveSoftware/steam-audio) används för att generera 3D-ljud och görs tillgänglig för användning under [Apache 2.0-licensen](https://github.com/ValveSoftware/steam-audio/blob/master/LICENSE.md).

Alla tredjepartsproduktnamn, företagsnamn och logotyper är varumärken eller registrerade varumärken och tillhör sina respektive innehavare.
