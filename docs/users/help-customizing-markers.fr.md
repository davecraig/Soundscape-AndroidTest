# Personnalisation de marqueurs

Si vous souhaitez renommer un marqueur créé précédemment ou lui ajouter une annotation, sélectionnez le marqueur dans l’onglet *"Marqueurs"* de la page *"Marqueurs et Itinéraires"*, puis appuyez sur le bouton *"Modifier le marqueur"*. Vous pouvez ainsi donner aux marqueurs des surnoms descriptifs ou utiles, et leur ajouter une description plus longue dans le champ d’annotation.

Sur cet écran *"Modifier"*, vous pouvez également supprimer un marqueur si vous n’en avez plus besoin.
